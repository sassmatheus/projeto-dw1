Requisitos

Nesta atividade vocês deverão criar no mínimo 3 páginas HTML. Isso fará com que vocês coloquem em prática o conhecimento de marcação HTML obtido até o momento, sendo assim suas páginas HTML deverão ter (no mínimo):

  • Títulos (H1, H2, H3, etc)
  • Parágrafos
  • Imagens com legendas
  • Links internos (bookmarks) e externos.
  • Marcações de formatação de texto (ênfase, negrito, etc)
  • Uma tabela com alguns dados
  • Uma lista (com ou sem ordem)

Para este trabalho os requisitos mínimos (propriedades que você DEVERÁ utilizar) são:

  • backgrounds (color, image, repeat, etc)
  • color, text-align, text-indent, text-decoration, line-height
  • font-family, font-style, font-weight, font-size
  • border, border-radius
  • width, height,
  • margin e padding
  • display e/ou float

Além disso há algumas regras:

  • Utilizar CSS externos
  • Utilizar no mínimo 5 famílias de fontes diferentes (3 do Google Fonts)
  • Utilizar unidades relativas em algum lugar.
  • Padronizar os tamanhos das imagens.
  • Posicionar algumas imagens ao lado de textos
  • Alterar cores de texto e plano de fundo de alguns elementos
  • Utilizar layouts flexíveis (Flexbox) em algum lugar.
